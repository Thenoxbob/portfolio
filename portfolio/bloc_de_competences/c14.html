C14 - Planification des activités

Formation Pro

Réalisation en cours de formations

🎯 Objectif

La compétence C14 consiste à planifier, organiser et suivre les activités d’un projet informatique afin de respecter les délais et les contraintes.
Dans le cadre de mon projet de BTS SIO, j’ai utilisé l’outil Trello pour gérer efficacement les tâches.

🛠️ Organisation de la planification

Pour organiser mon travail, j’ai mis en place un tableau sur Trello basé sur la méthode Kanban.
Le tableau est composé de trois colonnes principales :

  • À faire : tâches non commencées
  • En cours : tâches en cours de réalisation
  • Terminé : tâches finalisées
📋 Gestion des tâches

Chaque tâche du projet est représentée par une carte Trello.
Exemples de tâches :

  • Analyse des besoins
  • Conception
  • Développement
  • Tests
  • Documentation
⚙️ Procédure d’utilisation de Trello
  1. Créer un tableau pour le projet
  2. Ajouter les colonnes : À faire / En cours / Terminé
  3. Créer une carte pour chaque tâche
  4. Ajouter une description et des sous-tâches
  5. Définir une date d’échéance
  6. Déplacer les cartes selon l’avancement

👉 Cette procédure permet de suivre facilement le projet.

📊 Suivi du projet

Le suivi est réalisé en déplaçant les tâches :
À faire → En cours → Terminé

Cela permet de :

  • visualiser l’avancement
  • identifier les retards
  • ajuster le travail
⚠️ Difficultés rencontrées
  • Mauvaise estimation de certaines tâches
  • Retards dans le développement

Solutions :

  • Découpage des tâches
  • Mise à jour régulière du tableau
✅ Compétences acquises
  • Organisation du travail
  • Gestion du temps
  • Suivi de projet
  • Utilisation d’un outil professionnel
🧠 Bilan

L’utilisation de Trello m’a permis de mieux organiser mon projet, de suivre l’avancement et de respecter les délais.
