📘 Maintenance et amélioration d'une solution (Debian sur Azure)
🎯 Objectif

La compétence C17 consiste à maintenir une solution informatique en condition opérationnelle et à proposer des améliorations.

🧩 Contexte du projet
  • Projet : Déploiement d'un serveur Linux
  • Système : Debian
  • Plateforme : Microsoft Azure
  • Objectif : Héberger et maintenir un service accessible en ligne
🛠️ Réalisation

J'ai déployé une machine virtuelle sous Debian sur Microsoft Azure. J'ai ensuite assuré la maintenance du système :

  • Mise à jour du système
  • Installation des services nécessaires
  • Surveillance du serveur
  • Correction des problèmes
📋 Actions de maintenance
  • Mise à jour avec les commandes Linux (apt update / upgrade)
  • Vérification du bon fonctionnement des services
  • Surveillance des ressources (CPU, RAM)
  • Sécurisation du serveur (accès, utilisateurs)
⚙️ Procédure
  1. Créer une machine virtuelle sur Microsoft Azure
  2. Installer Debian
  3. Se connecter en SSH
  4. Mettre à jour le système
  5. Installer les services nécessaires
  6. Vérifier le bon fonctionnement
⚠️ Difficultés rencontrées
  • Problèmes de connexion SSH
  • Configuration du réseau
  • Gestion des droits utilisateurs

Solutions :

  • Vérification des ports
  • Configuration correcte des accès
  • Utilisation des commandes Linux
✅ Compétences acquises
  • Administration Linux
  • Gestion d'un serveur cloud
  • Maintenance informatique
  • Sécurisation d'un système
🧠 Bilan

L'utilisation de Debian sur Microsoft Azure m'a permis de :

  • ✔️ Maintenir un serveur en condition opérationnelle
  • ✔️ Améliorer la fiabilité du système
  • ✔️ Acquérir des compétences en administration système
