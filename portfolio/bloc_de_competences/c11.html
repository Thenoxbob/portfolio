📘 Publication d'un site avec GitHub
🎯 Objectif

La compétence C11 consiste à tester et mettre à disposition une solution.

🧩 Contexte
  • Projet : Publication d'un site web
  • Outil : GitHub
🛠️ Réalisation

J'ai utilisé GitHub pour :

  • Héberger le code du site
  • Publier le site en ligne (GitHub Pages)
📋 Étapes réalisées
  • Création d'un dépôt
  • Ajout des fichiers du site

⚙️ Procédure
  1. Créer un dépôt sur GitHub
  2. Ajouter les fichiers (HTML, CSS, JS)
  3. Faire un commit
  4. Activer GitHub Pages
  5. Tester le site en ligne
