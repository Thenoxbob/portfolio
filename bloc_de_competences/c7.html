📘 Mise en ligne de GLPI
🎯 Objectif

La compétence C7 consiste à déployer et rendre accessible une solution informatique.

🧩 Contexte
  • Projet : Mise en place de GLPI en ligne
  • Outil : GLPI
🛠️ Réalisation

J'ai installé et configuré GLPI sur un serveur afin de le rendre accessible en ligne.

📋 Étapes réalisées
  • Installation du serveur (Apache / PHP / MySQL)
  • Installation de GLPI
  • Configuration de la base de données
  • Accès via navigateur web
⚙️ Procédure
  1. Installer un serveur web (Apache)
  2. Installer PHP et MySQL
  3. Télécharger GLPI
  4. Configurer la base de données
  5. Accéder à GLPI via navigateur
⚠️ Difficultés rencontrées
  • Problèmes de configuration serveur
  • Accès à la base de données

Solutions :

  • Vérification des paramètres
  • Tests de connexion
✅ Compétences acquises
  • Déploiement d'application
  • Configuration serveur
  • Gestion d'un service web
🧠 Bilan

Cette activité m'a permis de comprendre comment rendre une application accessible en ligne.
